# 🚀 Hyperlocal Delivery Analytics

## 📌 Project Overview

Hyperlocal Delivery Analytics is an end-to-end Data Analytics project that analyzes the operational performance of India's leading quick-commerce platforms. The project integrates multiple public datasets into a unified analytical data model to evaluate delivery efficiency, inventory performance, customer experience, and product demand.

The primary objective is to simulate a real-world analytics workflow followed by Data Analysts and Analytics Engineers, covering the complete data lifecycle from Business Understanding to Dashboard Development.

---

## 🎯 Project Objectives

- Compare delivery performance across multiple quick-commerce platforms.
- Analyze product demand and inventory availability.
- Measure customer satisfaction through ratings and reviews.
- Evaluate delivery efficiency using SLA and delivery time metrics.
- Build a unified analytical data model from heterogeneous datasets.
- Develop an end-to-end ETL pipeline for data standardization.
- Create interactive Power BI dashboards for business insights.

---

## 🏙️ Project Scope

The analysis focuses on selected Bangalore pincodes to understand hyperlocal delivery operations across:

- Blinkit
- Zepto
- Swiggy Instamart

**Supporting Datasets**

- BigBasket Product Dataset
- Quick Commerce Benchmark Dataset

> **Note:** BigBasket and Quick Commerce datasets are used for benchmarking and additional analysis. They are **not directly merged** into the unified analytical model because they do not share common identifiers with the primary datasets.
