# Hyperlocal Delivery Analytics

## Project Overview

This project analyzes the operational performance of India's leading quick-commerce platforms:

- Blinkit
- Zepto
- Instamart
- BigBasket
- JioMart

The analysis focuses on selected Bangalore pincodes to understand:

- Delivery Performance
- Inventory Management
- Customer Satisfaction
- Product Availability
- Business Insights

## Tech Stack

- Python
- Pandas
- SQL
- Power BI
- Excel
- Git & GitHub
